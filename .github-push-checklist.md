## âš ï¸ Common Mistakes to Avoid

1. **Don't commit `.env` files** - Use `.env.example` instead
2. **Don't commit service account keys** - Keep them local only
3. **Don't hardcode credentials** - Use environment variables
4. **Don't commit `node_modules/`** - Let npm install handle it
5. **Don't commit build artifacts** - Add `dist/` and `build/` to `.gitignore`
